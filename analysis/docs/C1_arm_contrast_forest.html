8 Weeks: Whole-Bacterium Assays & Effector Functions — Maternal TdaP vs TT

At the infant baseline visit (8 weeks, before primary vaccination) this part tests whether maternal TdaP shifts the infant’s whole-bacterium responses and antibody effector functions relative to maternal TT (referent). It covers two families:

  • Whole-bacterium assays — SBA, WT_IgG, PTNA (serum bactericidal activity, whole-cell-binding IgG, and pertussis-toxin neutralization).
  • Effector functions binding to individual antigens — ADCD (all six antigens PT, FHA, PRN, FIM, TT, DT), and ADCP / ADNP (PT, PRN, TT, DT only — no FHA or FIM assays).

Each feature is a Wilcoxon rank-sum test of TdaP vs TT on log10 assay values — the same test used in the response-bubble grid. Multiplicity is handled by Benjamini–Hochberg within each assay class (Whole / ADCD / ADCP / ADNP as separate families). The forest plot additionally carries the point estimate + interval that natively pair with that test: the Hodges–Lehmann location shift and its Wilcoxon CI, oriented TdaP − TT. With TT as the reference level, this is the effect of TdaP relative to TT, so a positive value means higher with maternal TdaP.

🧭 Maps to interpretation: Infant-baseline (pre-primary-vaccination) read-out of the maternal vaccination effect on functional / whole-bacterium immunity, complementing the antigen×analyte bubble grid.

  • Q3 — Function vs binding: does the maternal effect reach functional read-outs (SBA/PTNA, ADCD/ADCP/ADNP), not just IgG binding?
  • Q2 — Antigen specificity: within each effector, is the effect concentrated in the vaccine antigens (PT/FHA/PRN) vs the toxoids (TT/DT)?

Notes on Method Choices

  • Same test as the bubbles. The p-values above come from wilcox.test() on log10 assay values, TT vs TdaP, exactly as in bubble_arm_effect(). Only the effect display differs: the bubbles show a bare difference of medians (no interval), whereas a forest plot needs one, so the point/CI use the Hodges–Lehmann shift and Wilcoxon conf.int — the estimator/interval that belongs to this test. The raw p is identical to the bubble p.
  • Orientation. Everything is oriented TdaP − TT with TT as the reference level, so the estimate is the effect of TdaP relative to TT (positive = higher with maternal TdaP), the sign convention of Saso et al. 2026 Figure 2C/D. The Direction column reuses the bubble labels (“Higher TdaP” / “Lower TdaP”). Magnitudes and p-values are identical to the bubble grid; only the displayed sign convention differs (the bubble reports median(TT) − median(TdaP)).
  • Multiplicity. Benjamini–Hochberg is applied within each assay class (Whole = 3, ADCD = 6, ADCP = 4, ADNP = 4 tests), so each functional family is corrected on its own. To correct across the whole panel instead, call infmon2_arm_effect(..., by_class = FALSE); to change method, set p_adjust_method.
  • Antigen coverage. ADCP and ADNP have no FHA or FIM assays, so those cells are absent by design (not missing data). ADCD spans all six antigens.
  • Infant arms pooled. This is the overall maternal effect at baseline. For an aP/wP-stratified version, pass infant_arm_sel = "aP" / "wP" to infmon2_arm_effect().
