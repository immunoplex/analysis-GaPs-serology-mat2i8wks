Phase 08 (Part A) — Decomposition and Predictors of the Maternal Low vs High Responder Categories

Do clinical covariates or a maternal serological signature predict which mothers are LOW vs HIGH responders? Stratified on the maternal arm (TdaP vs TT) only — no infant-arm analysis.

Scot Zens and Anja Saso

2026-07-13

Analytical Framework

Phase 07 Stage A split mothers into Low and High responders from their binding antibody across the two maternal draws (PregEarly + MatBirth; features PT_IgG, FHA_IgG, PRN_IgG). This branch asks a single question about that split:

Is there a consistent clinical covariate or maternal serological signature that predicts which mothers are Low vs High responders?

Two facts shape the whole analysis:

  1. The responder class is almost the vaccine arm. In Phase 07 the maternal Low/High split tracked the arm closely (TdaP mostly High, TT mostly Low). So the genuinely informative question is not “does anything predict the class” (the arm does) but “does anything predict it beyond the arm, and within an arm” — e.g. among vaccinated (TdaP) mothers, who still responds poorly?
  2. Infant priming (aP/wP) is not relevant here. This is a question about mothers; there is no infant-arm stratification anywhere in this document. The only stratifier is the maternal arm (TdaP vs TT).

The analysis runs in three parts:

  • Part A — the arm baseline. How much of the Low/High split is just the arm? This sets the bar every other predictor must clear.
  • Part B — clinical covariates. The ten clin_assess covariates, screened univariately (continuous + tertile forms) against the responder score (linear) and the Low/High class (logistic), overall and within each arm, then a maternal-arm-adjusted multivariable model.
  • Part C — serological signature. A maternal-serology feature matrix (all antigen × analyte features at the maternal visits, excluding the 3 features that define the score), screened by univariate AUC and modelled with an arm-adjusted elastic net (cross-validated AUC), with a clean pre-vaccination (PregEarly) baseline model.

Decomposition of the responder score categories

The Low/High split is the mean Blom-standardised maternal response across the three defining features (PT_IgG, FHA_IgG, PRN_IgG) at the maternal visits (PregEarly, MatBirth). All three of PT, FHA and PRN enter the score; no ACT or PENTAMER feature does (neither is measured in this dataset, and both are explicitly excluded from every serology step below). This section decomposes the categories into the contribution of each defining antigen, so the High/Low classes can be read in terms of the antibodies that build them.

.blom_d <- function(x) { r <- rank(x, na.last = "keep"); n <- sum(!is.na(x)); stats::qnorm((r - 3/8) / (n + 1/4)) }
dec <- data_raw %>%
  dplyr::filter(visit_name %in% MAT_VIS, as.character(feature) %in% RESP_FEATS) %>%
  dplyr::group_by(subject_accession, feature, visit_name) %>%
  dplyr::summarise(v = mean(log_assay_value, na.rm = TRUE), .groups = "drop") %>%
  dplyr::mutate(key = paste(feature, visit_name, sep = "@")) %>%
  dplyr::group_by(key) %>% dplyr::mutate(z = .blom_d(v)) %>% dplyr::ungroup() %>%
  dplyr::group_by(subject_accession, feature) %>%
  dplyr::summarise(antigen_z = mean(z, na.rm = TRUE), .groups = "drop")
decm <- dec %>% dplyr::inner_join(mat_cls[, c("subject_accession", "responder", "score")],
                                  by = "subject_accession")
per_ant <- decm %>% dplyr::group_by(feature) %>%
  dplyr::summarise(
    n         = sum(is.finite(antigen_z)),
    mean_Low  = round(mean(antigen_z[responder == "Low"],  na.rm = TRUE), 3),
    mean_High = round(mean(antigen_z[responder == "High"], na.rm = TRUE), 3),
    diff      = round(mean(antigen_z[responder == "High"], na.rm = TRUE) -
                      mean(antigen_z[responder == "Low"],  na.rm = TRUE), 3),
    AUC       = round(auc_binary(antigen_z, responder), 3),
    loading   = round(stats::cor(antigen_z, score, use = "complete.obs"), 3),
    p         = tryCatch(signif(stats::wilcox.test(antigen_z ~ responder)$p.value, 3),
                         error = function(e) NA_real_),
    .groups   = "drop") %>%
  dplyr::arrange(dplyr::desc(diff))
print(knitr::kable(per_ant,

Synthesis Questions to Address

  1. Is the responder split mostly the vaccine? Compare the arm-only AUC (Part A) to everything else. If clinical/serology AUCs barely exceed it, the Low/High class is essentially a vaccine-exposure label.
  2. Any clinical covariate beyond the arm? In the arm-adjusted logistic (Table B-adj) and the within-arm screens, does any covariate (gestational age at vaccine, vaccine→birth interval, BMI, parity, Hb, age) stay significant? Within TdaP mothers especially — what predicts a vaccinated mother nonetheless responding poorly?
  3. Consistent across arms or arm-specific? Does a predictor that matters in TdaP also matter in TT (a general “good responder” trait), or is it arm-specific (interacts with the vaccine)?
  4. Continuous vs threshold. For any significant continuous covariate, did the continuous or tertile form win, and by how much (linear vs threshold effect)?
  5. Serological signature beyond the 3 defining features. Does the elastic-net CV-AUC (Table C2) exceed arm-only? Which features (Table C1/C3) — other pertussis antigens (FHA/FIM), carrier antigens (DT/TT), IgG subclasses, Fc-receptor binding, or effector functions — carry it, and is the signal baseline (pre-vaccination) or only concurrent at birth?
  6. Baseline predictivity (the clean test). Does the pre-vaccination (PregEarly) baseline model predict who becomes a High responder? A positive baseline signal would mean responder status is partly a pre-existing maternal trait, not only a vaccine response.
  7. Mechanistic reading. Pull Parts A–C together: is “Low responder” best described as (a) a TT/exposure label, (b) a clinical phenotype, (c) a pre-existing serological trait, or (d) largely unpredictable from the measured covariates?
