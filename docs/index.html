Overview

This repository contains the analysis code for:

Pertussis immunisation in pregnancy shapes the functional antibody landscape before primary infant pertussis vaccination Saso, Zens et al. (in preparation)

The analysis uses longitudinal systems serology from up to 312 mother–infant dyads in the Gambian Pertussis Study (GaPs), a randomised 2×2 maternal–infant vaccine trial.

Data

Data are not included in this repository. See data/README.md for the full schema and data access information.

Pipeline

The analysis comprises 8 prespecified components plus a cross-validated block-ladder summary (CK), matching the manuscript sections exactly.

Component Manuscript section Topic
C0 Methods Data harmonisation & IgG standardisation
C1 Results A Maternal arm contrast at 8-week baseline
C2 Results B Antibody chain: production → transfer → decay
C3 Results C Maternal responder phenotype & predictors
C4a Results D–E Forward prediction → PTNA
C4b Results D–E Forward prediction → SBA
C4c Results D–E Forward prediction → WT IgG binding
C4d Results E Arm interaction & Deming EIV regression
C5 Results F Concurrent infant models
C6 Results J Responder phenotype as predictor of infant function
C7 Results H Effector pathway: IgG1 → FcγR → ADCD → SBA
C8 Results I Subclass balance & complement mediation
CK Results K Cross-validated block-ladder decomposition

Execution order

Components must be run in this order due to data dependencies: C0 → C1 → C2 → C3 → C4a/C4b/C4c (parallel) → C4d → C5 → C6 → C7 → C8 → CK
